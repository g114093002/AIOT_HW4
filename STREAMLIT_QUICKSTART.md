# 🚀 Streamlit 應用快速入門

## 📌 一分鐘快速開始

### 前置要求
```bash
pip install streamlit plotly pandas numpy
```

### 運行應用
```bash
streamlit run streamlit_app.py
```

### 打開瀏覽器
訪問 `http://localhost:8501`

## 🎨 自定義建議

### 修改主題顏色
編輯 `.streamlit/config.toml`:
```toml
[theme]
primaryColor = "#667eea"     # 主色
backgroundColor = "#f8f9fa"  # 背景
secondaryBackgroundColor = "#ffffff"
textColor = "#2c3e50"        # 文字
```

### 添加新頁面
```python
page = st.radio("選擇頁面", ["首頁", "新頁面"])

if page == "新頁面":
    st.markdown("# 新頁面")
    # 添加內容
```

### 添加側邊欄控件
```python
with st.sidebar:
    st.markdown("# 控制面板")
    selected = st.selectbox("選項", ["選項1", "選項2"])
```

## 📈 性能優化

### 1. 使用緩存
```python
@st.cache_data
def load_data():
    return pd.read_csv('data.csv')
```

### 2. 條件渲染
```python
if st.checkbox("顯示圖表"):
    st.plotly_chart(fig)
```

### 3. 列容器
```python
col1, col2 = st.columns(2)
with col1:
    st.write("左列內容")
with col2:
    st.write("右列內容")
```

## 🐛 常見問題

### Q: 應用無法啟動
**A:** 確保已安裝依賴
```bash
pip install -r requirements.txt
```

### Q: 頁面加載很慢
**A:** 添加緩存裝飾器
```python
@st.cache_data
def expensive_function():
    # 耗時操作
    pass
```

### Q: 圖表不顯示
**A:** 檢查數據並使用 `st.write()` 調試
```python
st.write(df)  # 查看數據
st.plotly_chart(fig)  # 顯示圖表
```
